# Utaina

```
Utaina mai ngā waka 
Ngā waka o te motu 
Tōia mai rā ki uta 
Ki te takoto ranga. 
Hiki nuku, hiki e 
Hiki rangi, rangi e 
Tēnā, tēnā rā 
koutou katoa.
```

## English translation

```
Pull in the canoes
The canoes of the land
Drag them ashore
To rest on the strand
Pull down, pull down
Pull up, up high
Acknowledgments,
To all of you.
```
